# Cyclistic-Bike-Share-Analysis-Annual-Members-vs-Casual-Riders

## Project Overview

This project was completed as part of the Google Data Analytics Professional Certificate Capstone. The objective was to analyze 12 months of Cyclistic bike-share trip data and identify behavioral differences between annual members and casual riders.
Using SQL in Google BigQuery, over 5.5 million ride records were cleaned, transformed, and analyzed to uncover trends in ride frequency, ride duration, weekday usage, seasonal demand, and bike type preferences. Tableau was used to create an interactive dashboard that communicates key insights and supports business decision-making.

## Business Problem

Cyclistic's marketing team wants to increase the number of annual memberships. To support this objective, this project answers the following business question:
**How do annual members and casual riders use Cyclistic bikes differently?**

## Tools Used

* Google BigQuery
* SQL
* Tableau
* Google Docs

## Key Findings

* Annual members completed **3.01 million rides**, accounting for the majority of total trips.
* Casual riders averaged **31.94 minutes per ride**, compared to **13.37 minutes** for annual members.
* Members primarily rode during weekdays, indicating commuter-oriented behavior.
* Casual riders were most active during weekends, suggesting leisure and recreational usage.
* Ridership peaked during summer months for both customer groups.

## Recommendations

1. Launch weekend-focused membership campaigns targeting casual riders.
2. Promote membership cost savings through targeted marketing.
3. Increase marketing efforts during peak riding seasons.
